# User Asset Drop Folder

Put optional images or short videos for future effects here.

Recommended files:

- `hero-1.jpg`, `hero-2.jpg`, `hero-3.jpg`: high-contrast images for image distortion or reveal effects.
- `texture-noise.png`: monochrome texture for glitch or grain overlays.
- `logo.png`: transparent logo or symbol if the homepage needs branded motion.
- `clip-1.mp4`: short muted visual loop for video-compositing experiments.

Current pages do not require any assets from this folder. They work without extra files.
